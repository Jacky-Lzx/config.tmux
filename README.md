## Installation

1. Check the external programs section below and make sure you have them installed.
2. Install [TPM](https://github.com/tmux-plugins/tpm)
   - `git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm`
3. Install plugins
   - Press `prefix + I` (capital i, as in Install) to fetch the plugin

### External Programs

- yq: used by the plugin `joshmedeski/tmux-nerd-font-window-name`
